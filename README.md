# docker-octopus-tentacle
Docker repo for a generic linux tentacle worker in docker with development tools preinstalled.

- [octokitpy](https://pypi.org/project/octokitpy/)
- [jq](https://stedolan.github.io/jq/)
- [requests](https://pypi.org/project/requests/)
- [sfdx@7.82.1-0](https://github.com/forcedotcom/cli/blob/main/releasenotes/README.md#5071-december-3-2020---cli-7821-0)
- [aws-iam-authenticator](https://github.com/kubernetes-sigs/aws-iam-authenticator)
- [ytt - k14s.io](https://carvel.dev/ytt/)
- [kubectl](https://kubernetes.io/docs/reference/kubectl/overview/)

- [Linux Tentacle](https://octopus.com/docs/infrastructure/deployment-targets/linux/tentacle)
